# Nestor

## Описание

Просто перепечатывает текст из файла.

## Запуск

1)

```cmd
pip install pyautogui
python main.py
```

2)или запуск из .exe файла. Билд нормального размера (~9MB).

3)или билд на своей машине:

```cmd
pip uninstall pyinstaller
pip install https://github.com/pyinstaller/pyinstaller/archive/develop.tar.gz
pyinstaller main.spec
```

## Использование

1)**Оконный режим**:

* Кнопка "Файл" позволяет выбрать файл, из которого печатать;
* Опция "Из буффера обмена" позволяет вместо файла перепечатать прямо из буффера обмена (скопированный текст);
* "Интервал печати" - задержка между набором отдельных символов;
* "Задержка перед печатью" - задержка перед началом набора текста. Нужна для выставления курсора в нужное место.

2)**Консольный режим**:

* Запуск без параметров открывает оконный режим;
* Параметры можно посмотреть используя флаг '--help' ('-h'): `main.exe -h`, `main.exe --help`;
* '--source [путь_к_файлу]' ('-s') - путь к файлу, из которого переписать текст;
* '--clipboard [1_или_0]' ('-clip') - переписать из буффера обмена;
* '--timeout [секунд]' ('-t') - задержка перед началом печати;
* '--interval [секунд]' ('-i') - интервал между набором символов.

*Например*:

```cmd
cd "D:/path/to/program/main.exe"
./main.exe -s D:/path/to/file/text.txt --timeout 6 -i 0.1
```

## Замечания

Не переписывает Unicode.

Перепечатывает в зависимости от раскладки. Чтобы набирались английские символы необходимо переключиться на соответствующую раскладку.
